# راهنمای دستیار هوش مصنوعی

## نصب و راه‌اندازی

### 1. تنظیمات اولیه (فقط مدیر)

1. به بخش **دستیار هوش مصنوعی** از منوی کناری بروید
2. روی **ویرایش** در کارت تنظیمات کلیک کنید
3. اطلاعات زیر را وارد کنید:
   - **ارائه‌دهنده**: یکی از موارد زیر را انتخاب کنید:
     - OpenAI (GPT-4, GPT-3.5)
     - Anthropic (Claude)
     - Google (Gemini)
     - Groq (Llama - رایگان و سریع)
   - **API Key**: کلید API خود را وارد کنید
   - **مدل**: مدل مورد نظر را انتخاب کنید
   - **فعال‌سازی**: دستیار را فعال کنید

### 2. دریافت API Key

#### OpenAI
- به [platform.openai.com](https://platform.openai.com) بروید
- API Keys → Create new secret key

#### Anthropic (Claude)
- به [console.anthropic.com](https://console.anthropic.com) بروید
- Settings → API Keys

#### Google Gemini
- به [makersuite.google.com](https://makersuite.google.com) بروید
- Get API Key

#### Groq (رایگان و سریع - توصیه می‌شود)
- به [console.groq.com](https://console.groq.com) بروید
- API Keys → Create API Key
- مدل پیشنهادی: `llama-3.3-70b-versatile`

## قابلیت‌های دستیار

### 1. خواندن اطلاعات
دستیار می‌تواند اطلاعات زیر را از سیستم بخواند:

- ✅ **موجودی انبار**: تعداد محصولات، ارزش کل، موجودی کم
- ✅ **خلاصه فروش**: کل فروش، تعداد سفارشات، میانگین
- ✅ **اطلاعات مشتریان**: نام، تلفن، بدهی، تاریخچه خرید
- ✅ **جستجوی محصولات**: بر اساس نام یا SKU
- ✅ **وضعیت مالی**: موجودی حساب‌ها، درآمد و هزینه

### 2. انجام عملیات
دستیار می‌تواند عملیات زیر را انجام دهد:

- ✅ **ایجاد مشتری جدید**
- ✅ **ثبت هزینه**
- (قابل توسعه برای عملیات بیشتر)

## نمونه سوالات

### سوالات اطلاعاتی:
```
- موجودی انبار چطوره؟
- فروش این ماه چقدر بوده؟
- کدوم محصولات موجودیشون کمه؟
- وضعیت مالی چطوره؟
- 10 مشتری برتر کی‌ان؟
- محصول X چند تا موجوده؟
```

### درخواست عملیات:
```
- یک مشتری جدید به نام علی رضایی با شماره 09123456789 بساز
- یک هزینه 500000 تومانی برای خرید لوازم اداری ثبت کن
- اطلاعات مشتری با شناسه XXX رو بهم بده
```

## ابزارهای در دسترس دستیار

| ابزار | توضیح | پارامترها |
|------|-------|-----------|
| `get_inventory_summary` | خلاصه موجودی انبار | `warehouseId` (اختیاری) |
| `get_sales_summary` | خلاصه فروش | `days` (پیش‌فرض: 30) |
| `get_customer_info` | اطلاعات مشتری | `customerId` |
| `search_products` | جستجوی محصولات | `query` |
| `get_financial_summary` | خلاصه مالی | `days` (پیش‌فرض: 30) |
| `create_customer` | ایجاد مشتری | `name`, `phone`, `email`, `address` |
| `record_expense` | ثبت هزینه | `amount`, `description`, `accountId`, `category` |

## توسعه ابزارهای جدید

برای اضافه کردن ابزار جدید، فایل `src/lib/ai-agent.ts` را ویرایش کنید:

```typescript
{
  name: 'tool_name',
  description: 'توضیحات ابزار',
  parameters: {
    param1: 'type - توضیحات',
  },
  execute: async (params) => {
    // کد اجرای ابزار
    return result;
  },
}
```

## نکات امنیتی

⚠️ **مهم:**
- API Key را در محیط امن نگهداری کنید
- فقط مدیران می‌توانند تنظیمات را تغییر دهند
- همه عملیات دستیار در دیتابیس ثبت می‌شود
- دستیار فقط به داده‌های مجاز دسترسی دارد
