# مشخصات پروژه TheMoak ERP

## خلاصه اجرایی

**TheMoak ERP** یک سیستم جامع مدیریت منابع سازمانی (ERP) است که برای مدیریت کامل کسب‌وکارهای کوچک و متوسط طراحی شده است. این سیستم با استفاده از فناوری‌های مدرن وب و رابط کاربری فارسی، تمامی جنبه‌های مدیریتی یک کسب‌وکار را پوشش می‌دهد.

---

## ۱. اطلاعات کلی پروژه

### ۱.۱ نام پروژه
**TheMoak ERP** (سیستم مدیریت منابع سازمانی TheMoak)

### ۱.۲ نسخه
**0.1.0** (نسخه اولیه - در حال توسعه)

### ۱.۳ تکنولوژی‌های اصلی
- **Frontend Framework**: Next.js 14.2.3 (App Router)
- **UI Framework**: React 18.3.1
- **Styling**: Tailwind CSS 3.4.18
- **Database**: PostgreSQL با Prisma ORM 5.22.0
- **Authentication**: NextAuth.js 5.0.0-beta.30
- **Type Safety**: TypeScript 5
- **Date Handling**: moment-jalaali (تقویم شمسی)
- **UI Components**: Radix UI + shadcn/ui
- **State Management**: Zustand 5.0.8
- **Data Fetching**: TanStack Query 5.90.10
- **Drag & Drop**: @dnd-kit
- **Charts**: Recharts 3.5.1
- **Notifications**: Sonner 2.0.7

---

## ۲. معماری سیستم

### ۲.۱ معماری کلی
سیستم بر اساس معماری **Server Components** و **Client Components** در Next.js 14 ساخته شده است:

- **Server Actions**: تمام عملیات‌های دیتابیس و منطق کسب‌وکار در سمت سرور
- **Client Components**: کامپوننت‌های تعاملی و رابط کاربری
- **API Routes**: برای احراز هویت و یکپارچه‌سازی‌های خارجی

### ۲.۲ ساختار دیتابیس
- **Database**: PostgreSQL
- **ORM**: Prisma
- **Migration Management**: Prisma Migrations
- **Schema Location**: `prisma/schema.prisma`

## ۵. ساختار دیتابیس

### ۵.۱ مدل‌های اصلی

#### User (کاربر)
- اطلاعات کاربران سیستم
- نقش‌ها و دسترسی‌ها
- لاگ فعالیت‌ها

#### Account (حساب)
- حساب‌های بانکی و نقدی
- موجودی حساب‌ها
- پشتیبانی از چند ارز

#### Transaction (تراکنش)
- تراکنش‌های مالی
- انواع: درآمد، هزینه، انتقال، تعدیل
- ارتباط با حساب‌ها و پروژه‌ها

#### Product (محصول)
- اطلاعات محصولات
- قیمت خرید و فروش
- انواع: فروختنی، دارایی ثابت، مصرفی
- ارتباط با WooCommerce

#### Inventory (موجودی)
- موجودی محصولات در انبارها
- ردیابی چند انباره

#### Order (سفارش)
- سفارشات فروش
- وضعیت پرداخت و تحویل
- ارتباط با مشتریان

#### PurchaseOrder (سفارش خرید)
- سفارشات خرید از تامین‌کنندگان
- گردش کار کامل
- مدیریت هزینه‌های اضافی

#### FixedAsset (دارایی ثابت)
- دارایی‌های ثابت و مصرفی
- محاسبه استهلاک
- روش‌های استهلاک

#### Project (پروژه)
- پروژه‌ها و وظایف
- مدیریت بودجه
- وضعیت پروژه‌ها

#### Customer (مشتری)
- اطلاعات مشتریان
- اعتبار و شرایط پرداخت
- ارتباط با CRM

## ۹. نصب و راه‌اندازی

### ۹.۱ پیش‌نیازها
- Node.js >= 18.17.0
- PostgreSQL
- npm یا yarn

### ۹.۲ نصب
```bash
npm install
```

### ۹.۳ تنظیمات
1. کپی فایل `.env.example` به `.env`
2. تنظیم `DATABASE_URL`
3. تنظیم `NEXTAUTH_SECRET`
4. اجرای migrations:
```bash
npx prisma migrate dev
```

### ۹.۴ اجرا
```bash
npm run dev
```
